# Cognify — Como rodar o projeto

## 1. Instalar

Abra o terminal na pasta do projeto e rode:

```bash
npm run install:all
```

Copie o arquivo de configuração da API:

```bash
cp api/.env.exemple api/.env
```

No Windows, copie manualmente `api/.env.exemple` para `api/.env`.

## 2. Rodar

```bash
npm run dev
```

## 3. Acessar

- Frontend: http://localhost:5173
- API: http://localhost:3000
